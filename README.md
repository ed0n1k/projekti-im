# 💈 Smart Barber Appointment System

A web application designed to help barbershops manage appointments, clients, and services online.  
The system allows customers to easily book appointments while barbers can manage schedules and services efficiently.

---

## 📌 Project Overview

Many small businesses such as barbershops, beauty salons, and clinics in Kosovo still rely on phone calls or walk-ins for appointments.

This project aims to provide a **modern web-based solution** that allows:

- Online appointment booking
- Schedule management
- Client management
- Appointment reminders
- Product marketplace

---

## 🎯 Main Features

### 👤 Client
- Register and login
- Book appointments online
- Pay a small booking deposit (10–20 cents)
- Receive appointment confirmation
- View appointment history
- Buy barber products from the marketplace

### 💼 Staff (Barber)
- View upcoming appointments
- Manage working schedule
- Manage services
- View client information

### 👑 Admin
- Add or remove staff members
- Manage services and pricing
- View business statistics
- Manage marketplace products

---

## 🛒 Marketplace

The application includes a small marketplace where clients can purchase barber-related products such as:

- Hair gel
- Beard oil
- Hair wax
- Shampoo
- Grooming products
